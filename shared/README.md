## features

- 📊 Stats - output bag bundlesize
- 📦 exclude `dependencies` and `peerDependencies`
- 💪 Typescript - type safe

## install

```console
pnpm add @aiou/rollup-template
```

## development

- **Setup** - `pnpm i`
- **Build** - `pnpm build`
